# Jenkins Plugins
*   **Git Plugin**
    *   Used for Source Code Management (SCM) to pull code from repositories like GitHub, GitLab, or Bitbucket.
*   **Pipeline**
    *   The core suite of plugins required to create Declarative and Scripted pipelines in Jenkins.
*   **SSH Build Agents**
    *   Used to connect the Jenkins Master server to remote Agent/Slave nodes via SSH for distributed builds.
*   **Maven Integration**
    *   Used to configure the Maven build tool globally in Jenkins (defining `MAVEN_HOME` and versions).
*   **GitHub Plugin**
    *   Used specifically for GitHub integration, such as triggering builds automatically via Webhooks.
*   **GitLab Plugin** (Mentioned as an alternative to GitHub)
    *   Used for integration with GitLab repositories.
