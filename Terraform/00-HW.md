Assignment & Course Strategy
Instructor Note:

Homework Assignment: The "Deploy Three tier" 
S3: For the Frontend.
EKS: For the Backend.
RDS (Relational Database Service): For the Database 
Requirements: Place these components in your modules folder. Use .tfvars files for variables.
Scope: You need to write the complete scope for a multi-environment infrastructure (Dev, Test, and Prod).
Advice: Avoid using ChatGPT excessively. If you rely on it too much, your logic-building skills and "brain muscles" will not develop. Use public documentation for the majority of your work.
